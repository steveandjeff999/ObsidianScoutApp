## Command Reference

### MVVM Toolkit Command Generation

When you add `[RelayCommand]` to a method:
```csharp
[RelayCommand]
private async Task RefreshAsync()
```

The toolkit generates:
```csharp
public IAsyncRelayCommand RefreshAsyncCommand { get; }
```

**Naming Rule**: `{MethodName}Command`
- `RefreshAsync()` ? `RefreshAsyncCommand`
- `LoadMatches()` ? `LoadMatchesCommand`
- `ExportJson()` ? `ExportJsonCommand`
