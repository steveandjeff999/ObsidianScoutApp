## Real-World Scenarios

### Scenario 1: Alliance Selection
**Goal:** Choose best alliance partners

**Use:** ?? **Team Averages**
```
1. Select all potential partners
2. Choose "Team Averages"
3. Compare total points
4. Check consistency (±value)
5. Pick highest average + lowest deviation
```

**Why:** Quick comparison, overall performance

---

### Scenario 2: Team Improvement Tracking
**Goal:** See if team is getting better

**Use:** ?? **Match-by-Match**
```
1. Select your team only
2. Choose "Match-by-Match"
3. Generate line chart
4. Look for upward trend
5. Identify what's working
```

**Why:** Shows progression over time

**Goal:** Find most reliable teams


## Quick Troubleshooting

### Charts Show No Data
```
Check: Do teams have scouting data?
? Go to Teams page
? Check "Scouting Data Count" column
? If 0, teams have no data for this event
```

### Only One Data Point Shows
```
Check: Is "Team Averages" selected?
? Switch to "Match-by-Match"
? Regenerate graphs
? Should show multiple points
```

### View Switching Doesn't Work
```
Check: Output window for errors
? Look for "Data View: match_by_match"
? If missing, command not firing
? Try rebuilding app
```

## Performance Tips

### For Fast Loading:
1. **Select fewer teams** (1-3 teams load faster)

### For Detailed Analysis:
1. **Start with Match-by-Match**
2. **Use Line Chart** for trends
3. **Check individual outliers**
4. **Compare against Team Averages**
